# Drum Kit Project

## Overview
An interactive drum kit built with images and sounds. Each drum component—snare, bass, toms, hi-hat, and cymbals—is represented visually, and when clicked or triggered, it plays its unique sound. This project provides a fun, engaging way to explore rhythm and percussion through a simple web interface.

## Features
- 🎵 **Interactive Sounds:** Authentic drum sounds linked to each image.
- 🥁 **Visual Representation:** Clear images for every drum component.
- ⌨️ **Keyboard Support:** Play drums using assigned keys for faster interaction.
- 🌐 **Lightweight & Accessible:** Runs directly in the browser with no extra setup.
